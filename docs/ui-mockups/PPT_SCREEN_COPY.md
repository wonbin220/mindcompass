# 주요 화면 설명 문구

| 화면명 | 화면 목적 | 주요 기능 | 핵심 포인트 |
|---|---|---|---|
| 로그인 | 사용자가 앱에 안전하게 진입하는 시작 화면 | 이메일 로그인, 비밀번호 입력, 회원가입 이동 | 첫 진입 장벽은 낮추고 서비스 신뢰감을 주는 구조 |
| 캘린더 홈 | 월간 감정 흐름과 날짜별 기록 여부를 확인하는 메인 화면 | 월간 감정 표시, 날짜 선택, 일기 요약 확인, 작성/상세 이동 | 기록 유무를 직관적으로 보여주고 일기 작성으로 자연스럽게 연결 |
| 일기 작성/상세 | 하루의 감정과 생각을 기록하고 AI 분석 결과를 확인하는 화면 | 제목/본문 작성, 대표 감정/강도 선택, AI 분석/위험도 확인, 채팅 연결 | 기록은 간단하게, 저장 후에는 해석과 다음 행동으로 이어지도록 구성 |
| 채팅 | 사용자가 AI와 대화를 통해 정서적 지원을 받는 화면 | 세션 생성, 메시지 송수신, 일반/지원형/안전형 응답 분기, 대화 이력 저장 | 사용자 상태에 따라 NORMAL, SUPPORTIVE, SAFETY 응답을 구분 제공 |
| 리포트 | 기록 데이터를 바탕으로 감정 패턴과 위험도 흐름을 확인하는 화면 | 월간 요약, 주간 감정 추이, 월간 위험도 추이, 대표 감정 확인 | 단순 통계가 아니라 사용자가 자신의 상태를 이해하고 회고할 수 있게 구성 |
